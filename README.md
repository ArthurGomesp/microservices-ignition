# Ignição Microsserviços 🚀

Projeto baseado no curso **Ignição Microsserviços**, com foco em construção de aplicações usando arquitetura de microsserviços e boas práticas de desenvolvimento.

---

## 📚 Conteúdo

* Arquitetura de microsserviços
* Domain-Driven Design (DDD)
* APIs REST e Domain Services
* Persistência com Spring Data JPA
* Comunicação entre serviços (REST e eventos)
* Testes de integração com REST Assured
* Event-Driven Architecture com Kafka
* Padrões de resiliência

---

## ⚙️ Tecnologias

* Java
* Spring Boot
* Spring Data JPA
* REST Assured
* Apache Kafka

---

## 🎯 Objetivo

Praticar conceitos de arquitetura moderna, integração entre serviços e construção de sistemas distribuídos.

---

## 📌 Observação

Projeto com fins educacionais.
